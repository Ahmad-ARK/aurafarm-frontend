Abstract

Smallholder tomato farmers in Pakistan face persistent crop losses due to late disease detection, unguided fertilizer use, and the absence of affordable, localized advisory tools. AuraFarm is a bilingual (Urdu & English) Progressive Web Application that delivers three core capabilities on a smartphone browser: CNN-based leaf disease detection across 10 tomato disease classes using a ResNet model hosted on Modal.com, a weather-driven Disease Severity Value (DSV) accumulation algorithm that generates automated CRITICAL / HIGH / WARNING alerts nightly from Open-Meteo forecast data, and a soil-aware NPK fertilizer calculator that adjusts recommendations for pH, electrical conductivity, organic carbon, and water source.

Designed for low-literacy users, AuraFarm renders all content in Nastaliq Urdu (Gulzar typeface) with full right-to-left layout. The system manages complete crop cycles across four growth stages and tracks GPS-pinned farm plots. The backend runs on Express.js with PostgreSQL via Prisma ORM, deployed on Railway; the frontend is a Next.js 14 PWA with zero-reload CSS Keep-Alive navigation.

Problem Statement

Pakistani smallholder farmers lose an estimated 30–40% of their tomato crop annually to preventable diseases and suboptimal fertilization, yet existing digital agricultural tools are English-only, require paid subscriptions, or assume desktop access.

AuraFarm bridges this gap by providing a free, mobile-first, Urdu-first precision agriculture assistant that operates on any smartphone browser — requiring no app store installation — and surfaces AI-driven insights in the farmer’s native language with locally relevant product recommendations.

Objectives
  • Develop a bilingual PWA with full Nastaliq Urdu support, RTL layout, and Gulzar typeface rendering for low-literacy accessibility.
  • Implement a 10-class CNN tomato leaf disease detector with Urdu result presentation and local Pakistani fungicide/pesticide recommendations.
  • Build a nightly DSV weather-driven disease alert engine using real-time Open-Meteo forecast data with tunnel farming risk multipliers.
  • Create a soil-aware NPK fertilizer calculator incorporating pH correction, EC salt-stress penalties, organic carbon bonuses, and previous-crop nitrogen credit.
  • Enable GPS-pinned multi-plot farm management with automated crop stage tracking across four growth phases from transplant to harvest.

AuraFarm successfully delivers an end-to-end precision agriculture stack accessible from any smartphone browser. The DSV algorithm correctly suppresses duplicate alerts through an in-run seen-set, while the CSS Keep-Alive navigation eliminates redundant API calls on tab switching. The CNN model, served via Modal.com serverless GPU, returns disease predictions with confidence scores and Urdu recommendations including locally available Pakistani product names (Dithane M-45, Ridomil Gold, Kocide 2000, Confidor, Admire) — a critical differentiator for field usability.

All three core modules (disease detection, weather alerts, fertilizer planning) operate independently and can function for any active crop cycle, making the system usable from day one of the nursery period through final harvest.
